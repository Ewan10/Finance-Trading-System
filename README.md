# Finance Trading System

## Description

A Finance trading system for managing and executing financial transactions of financial products and assets implemented in Angular (Presentation Layer), Spring/Java (Business Layer) and C++ (Engine).

Components of crucial functionality of the engine will be the limit order book, the matching engine and others. The Spring API will function as an intermidiary between engine and client. The client is implemented in Angular and raw css will be used for the styles for more self-practice applying responsive design principles as using the appropriate units, rem, media queries and flex box or grid whenever is necessary.
